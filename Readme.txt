Цель приложения - создавать/хранить/редактировать деклисты, которыми я играю в MTG

1. Начал делать пару дней назад, над вьюхами не заморачивался пока. Единственное, на что есть смысла нажимать - две ссылки внизу главной страницы.

2. Валидации считай что нет.

3. Карты можно добавлять отдельно (AddCard), имена должны быть уникальными или с формы AddDeck добавятся автоматически.

4. На AddDeck можно пока добавить только три карты, следующий шаг - динамически добавлять строку при добавлении следующей карты.

5. В базе деклист хранится не единой сущностью, а как набор связок Карта-Дека-Количество (да, это дебильный вариант и можно сделать намного проще, но во-первых я хочу потом отслеживать историю изменений по индексу, а во-вторых, это дает мне возможность решать нетривиальные задачи).

6. База в самом начале содержит только форматы, все остальное пусто.
